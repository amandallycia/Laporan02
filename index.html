1. Mengapa menggunakan CSS dan JavaScript dalam Pemrograman Web?

Apakah Anda ingin menutup jawaban ini?

→ Berdasarkan dengan yang telah saya pelajari pada perkuliahan minggu-minggu sebelumnya saya dapat berpendapat bahwa dalam pemograman web penggunaan CSS dan JavaScript sangat penting dikarenakan keduanya mempunyai perannya masing-masing untuk pemrograman web dengan cara saling melengkapi. CSS atau singkatan dari Cascading Style Sheets biasanya digunakan untuk mengatur tampilan dan desain dari halaman web misalnya seperti warna, jenis huruf, tata letak, margin, dan responsivitas terhadap berbagai ukuran layer. Dengan menggunakan CSS pada halaman yang ingin kita buat, tampilan website bisa menjadi lebih menarik. Sedangkan JavaScript digunakan untuk menambah interaktivitas pada halaman web yang ingin kita buat misalnya seperti tombol yang bisa diklik, animasi, serta elemen-elemen yang bisa berubah secara dinamis tanpa perlu me-refresh halaman lagi. Kesimpulannya adalah, CSS dan JavaScript masing-masing mempunyai peran penting dalam pemrograman web karena dengan memakai keduanya dapat membuat web lebih menarik dan interaktivitas yang dinamis.

2. Bagaimana CSS dan JavaScript dapat diterapkan dalam Pemrograman Web?

Apakah Anda ingin menutup jawaban ini?

→ Terdapat 2 cara utama untuk mengintegrasikan CSS dan JavaScript ke dalam file HTML, diantara yaitu ad acara internal dan cara eksternal. Untuk secara internal, CSS dapat langsung dituliskan dalam tag <style> pada bagian <head> dalam dokumen HTML, sedangkan untuk JavaScripct dapat dituliskan dalam tag <script> yang biasanya sering ditempatkan dibagian <head> atau sebelum penutup tag <body>. Sedangkan untuk secara eksternal, CSS dan Java bisa ditulis dalam file terpisah dengan ekstensi .css dan .js lalu dihubungkan ke file HTML menggunakan tag <link> untuk CSS dan tag <script> untuk JavaScript sendiri.

3. Jelaskan apa saja keuntungan menggunakan JavaScript dan CSS?

Apakah Anda ingin menutup jawaban ini?

→ Berikut adalah apa-apa saja keuntungan dari JavaScript dan CSS:

JavaScript: JavaScript sendiri memberikan kemampuan untuk menambahkan interaktivitas pada halaman web yang ingin dibuat, seperti animasi, validasi form sebelum dikirim ke server, manipulasi elemen secara dinamis, mendukung framework dan library besar, kompatibel dengan semua browser modern karena bisa digunakan pada sisi klien maupun sisi server, serta interaksi pengguna tanpa harus selalu me-reload halaman.

CSS: Adapun keuntungan yang didapat ketika memakai CSS adalah CSS bisa memberikan kemudahan dalam mengatur dan memperindah tampilan halaman web contohna seperti pengaturan warna, font, tata letak, dan responsivitas, sehingga desain website yang dibuat menjadi lebih menarik di setiap halamannya. CSS juga memungkinkan pengembangan untuk membuat tampilan yang lebih responsive dan bisa juga menyesuaikan dengan berbagai ukuran layer perangkat.

4. Jelaskan apa saja keuntungan menggunakan Bootstrap?

Apakah Anda ingin menutup jawaban ini?

→ Bootstrap mempunyai banyak keuntungan untuk pengembangan web, hal ini dikarenakan Bootstrap merupakan framework yang menyediakan macam-macam komponen siap pakai, contohnya seperti tombol, form, navigasi bar, card, sampai carousel, sehingga mempercepat dan lebih mengefesienkan proses pembuatan tampilan pada web. Selain itu, Bootstrap juga mempunyai system grid yang responsive yaitu fitur yang memudahkan pengembangan dalam saat Menyusun layout halaman secara otomatis yang menyesuaikan ukuran layar macam-macam perangkat. Bootstrap juga tergolong sangat mudah digunakan karena cukup dengan menerapkan class-class tertentu pada elemen HTML dan menghasilkan tampilan yang rapih dan professional tanpa perlu menulis bayak code CSS secara manual. Ditambah lagi Bootstrap juga memiliki dokumentasi yang lengkap dan komunitas yang luas sehingga dapat memudahkan pengembangan saat menggunakannya.

5. Contoh penggunaan Bootstrap

Apakah Anda ingin menutup jawaban ini?

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Contoh Tombol Bootstrap</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body class="text-center mt-5">
  <h1>Contoh Tombol Bootstrap</h1>
  <button class="btn btn-primary">Klik Saya</button>
</body>
</html>

→ Code HTMLdi atas digunakan untuk menampilkan sebuah halaman web sederhana yang berisi judul dan sebuah tombol dengan tampilan menarik. Di bagian atas ada judul besar yang ditulis dengan tag <h1> dan berisi teks “Contoh Tombol Bootstrap”. Tepat di bawah judul itu, ada sebuah tombol yang dibuat dengan tag <button>. Tombol ini diberi class btn-primary. Nah, btn itu artinya kita mau pakai gaya tombol dari Bootstrap, dan btn-primary bikin tombolnya berwarna biru yaitu warna khas tombol utama dalam desain Bootstrap. Karena di atas pakai Bootstrap, jadi ga perlu atur warna, ukuran, atau efek tombol sendiri. Semuanya udah otomatis. Singkatnya, dengan kode itu, bisa bikin halaman web sederhana yang tampilannya bersih, terpusat, dan tombolnya udah kelihatan profesional, cuma dengan beberapa baris kode aja. Dan ketika diklik bagian ”Klik Saya” nanti akan menuju ke halaman yang ingin dituju dan dibuat.
